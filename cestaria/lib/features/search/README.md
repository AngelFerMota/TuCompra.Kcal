# Search (Búsqueda)

Contiene UI, providers y lógica para:
- Búsqueda de productos (texto/código de barras)
- Sugerencias/autocompletado
- Integración con MercadonaApi y OpenFoodFactsApi

Estructura sugerida:
- `presentation/` widgets y páginas
- `providers/` Riverpod providers
- `controllers/` o `notifiers/`
- `repositories/` orquestación de datos
